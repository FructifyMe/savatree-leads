Who to Target, Why They'll Buy, and How Satellite Data Finds Them
SavATree — Jackson's Territory: Melrose, Stoneham, Wilmington, Wakefield, Reading, North Reading

The Big Picture: Think Contracts, Not Calls

Key insight: A single residential tree removal is a $500–$3,000 one-time job. An HOA or commercial property maintenance contract is $5,000–$50,000+/year in recurring revenue. The satellite analysis approach is dramatically more powerful for larger properties because: bigger canopy footprint = easier to detect from space, bigger liability exposure = stronger sales pitch, and the decision-maker has a budget and a legal obligation to act.

Here's how each property type stacks up as a lead — scored on contract value, likelihood to convert, satellite detectability, and urgency drivers.

Why #1: HOAs are legally obligated to maintain common area trees. Board members face personal liability exposure for negligence. Proposed Lead Scoring Model

Every flagged property gets a composite score (0–100) based on four weighted factors:

Factor Weight What It Measures Data Source
Canopy Decline Severity 30% NDVI drop magnitude over 2–3 year window. >20% drop = high score. Sentinel-2 / NAIP analysis
Property Classification 30% S-tier (HOA/commercial) = 30pts, A-tier = 22pts, B-tier = 15pts, C-tier = 8pts MassGIS land use codes + assessor data
Liability Exposure 25% Pedestrian areas, parking lots, proximity to buildings, public-facing. Higher foot traffic = higher score. Aerial imagery + land use classification
Property Value 15% Higher assessed value = more likely to invest in professional tree care. Also indicates ability to pay. Town assessor records
Example scoring: A 40-unit condo complex in Wakefield with 25% NDVI decline on common area trees near a parking lot, assessed at $8M total: Canopy Decline (25/30) + Property Class (30/30) + Liability (22/25) + Value (13/15) = 90/100 — hot lead, call immediately.

Revised MVP Strategy — Go Big First

Based on this analysis, the satellite analysis MVP should start with the biggest targets, not individual homes:

Phase 1 (Week 1–2): HOAs, Condos, and Commercial
  • Identify all HOA/condo communities and commercial properties in the 6 target towns using MassGIS land use data
  • Run Sentinel-2 NDVI change detection on these larger properties (they're big enough for reliable detection)
  • Score and rank the top 20–30 flagged properties
  • Use Apollo/Clay to find decision-makers: HOA board presidents, property managers, facility managers
  • Jackson does targeted outreach with satellite imagery as a conversation starter
Phase 2 (Week 3–4): Multifamily and Institutional
  • Expand to apartment complexes, schools, churches, assisted living
  • Same NDVI workflow, slightly smaller footprints
  • These leads often come through property management companies — use Apollo to find the management firm, not the building owner
Phase 3 (Month 2+): High-Value Residential
  • Only pursue residential leads where NAIP confirms large-lot properties with significant canopy
  • Filter to homes assessed >$600K with >0.5 acres
  • These are supplemental volume, not the core strategy
